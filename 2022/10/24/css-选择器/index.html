一. css 选择器的定义

css 选择器是用来选择标签的，选出来以后给标签加样式。

二. css 选择器的种类

  1. 标签选择器
  2. 类选择器
  3. 层级选择器(后代选择器)
  4. id选择器
  5. 组选择器
  6. 伪类选择器
